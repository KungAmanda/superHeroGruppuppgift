# superHeroGruppuppgift
## Installation
**Kopiera GitHub-länken och klona repositoriet till din lokala dator.**
**Öppna den klonade projektmappen i Visual Studio.**
**Öppna ditt föredragna databashanteringsverktyg (t.ex. SQL Server Management Studio).**
**Välj databasen och hitta tabellen "dbo.superheros"och "dbo.superheroTeam". Högerklicka på tabellen och välj "Visa data".**
**Fyll i önskade värden för superhjältarna som du vill ha med i tabellen och spara ändringarna.**
**Öppna package manager och skriv "update-database" **

### Andvändning 
**För att skapa en ny superhjälte, klicka på "Skapa" och fyll i formuläret med relevant information.**
**För att visa detaljer eller redigera en befintlig superhjälte, klicka på dess namn i listan och välj önskad åtgärd.**
**För att ta bort en superhjälte, klicka på "Ta bort" bredvid dess namn i listan.**
